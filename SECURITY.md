# Security Policy

## Supported Versions

Currently, only the latest version of CoreAmp is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

If you've found a security vulnerability in CoreAmp, please do not open a public issue. Instead, please report it privately.

**To report a vulnerability:**
1.  Submit an email to the project maintainer.
2.  Provide a detailed description of the vulnerability and steps to reproduce it.
3.  We will respond within 48 hours and work with you to resolve the issue.

Thank you for helping us keep CoreAmp secure!
